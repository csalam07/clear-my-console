# Clear-My-Console

> CMC is a Cross platform console which clear your Node.js CLI.

<br />

## Installation

### npm

```sh
npm install clear-my-console
```

### Yarn

```sh
yarn add clear-my-console
```
<br />

## Usage

```js
const cmc = require('clear-my-console');
// Clears the console.
cmc();
```
